ScreenSnap — 전문 스크린샷, GIF 녹화 및 주석 도구

웹 페이지의 모든 부분을 캡처, 녹화, 주석 처리. 4가지 캡처 모드, 풍부한 주석 편집기, 스크린샷 기록 관리 — 100% 로컬 실행, 데이터 수집 제로.

캡처 모드
- 영역 선택 (Alt+Shift+S): 드래그로 원하는 영역 선택, 화살표 키로 미세 조정
- 보이는 영역 (Alt+Shift+A): 한 번 클릭으로 현재 화면 캡처
- 전체 페이지: 자동 스크롤로 전체 페이지 결합, 고정 헤더 자동 제거
- GIF 녹화 (Alt+Shift+G): 선택 영역을 최대 15초 애니메이션 GIF로 녹화

주석 편집기
- 도형: 사각형, 둥근 사각형, 점선 사각형, 타원
- 화살표: 단방향/양방향 화살표, 직선
- 그리기: 자유 펜, 형광펜
- 텍스트: 글꼴 크기, 굵게, 기울임, 색상 사용자 지정
- 말풍선: 부드러운 꼬리의 대화 말풍선, 텍스트에 자동 맞춤
- 스탬프: 체크, 별, 이모지 등 빠른 표시
- 번호: 순서 번호 마커
- 모자이크 / 블러: 민감한 정보 숨기기
- 스포트라이트: 배경을 어둡게 하여 초점 영역 강조
- 돋보기: 세부 사항 확대
- 자르기: 캡처 후 다시 프레이밍
- 완벽한 실행 취소/다시 실행, 모든 도구에 키보드 단축키

기능
- 스크린샷 기록: 썸네일, 검색, 유형 필터 (이미지/GIF)
- PNG, JPG, WebP 저장 (품질 조절 가능), GIF 직접 다운로드
- 한 번 클릭으로 클립보드에 복사
- 우클릭 메뉴로 빠른 캡처
- 기본 색상 및 선 굵기 사용자 지정
- 줌 컨트롤 (마우스 휠 지원)
- 도구 모음 그룹 접기 — 깔끔하고 컴팩트한 UI
- 7개 언어 지원: 한국어, English, 中文简体, 中文繁體, 日本語, Español, Português
